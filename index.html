Configuración de TravelBot LiveChat

Configura tu LiveChat en Minutos

Configura el widget de TravelBot LiveChat con facilidad. Personaliza su apariencia, comportamiento e integraciones para mejorar la interacción con los clientes.

Comenzar

Código de Integración

Copia y pega el siguiente código en el <body> de tu sitio web para activar el widget de TravelBot LiveChat. 


<script id="live-chat-travelbot" src="https://widget.mychatbot.es/assets/js/widget.js" type="module"></script>
  document.addEventListener("DOMContentLoaded", () => {
    LiveChat.init({
      agent_id: "agent-id",
      position: "right",
    });
  });
</script>

Personalizar Apariencia

Establecer Reglas de Comportamiento

Integrar y Publicar

Conecta TravelBot con tu CRM, correo electrónico u otras herramientas. Copia el fragmento de código generado y pégalo en tu sitio web para activarlo.
